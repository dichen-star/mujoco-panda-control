# Day 8 日志 - 动力学方程基础

## 完成
- 实现 M(q), C(q,q̇), G(q) 三大动力学量的提取函数
- 验证反向/正向动力学一致性（机器精度 5e-15）
- 重力补偿曲线可视化（joint2 扫描）

## 实验结果

### 实验 1: M(q) 和 G(q) 随姿态变化
| 姿态 | M[0,0] (kg·m²) | ||G(q)|| (N·m) |
|------|---------------|----------------|
| HOME | 0.63 | 22.5 |
| Slight extension | 1.90 | 41.6 |
| Rotated sideways | 1.90 | 41.6 |
| More curled | 0.65 | 23.5 |

### 实验 2: 科氏力的速度二次关系
| 速度 (rad/s) | ||C·q̇|| (N·m) |
|--------------|---------------|
| 0.1 | 0.036 |
| 1.0 | 3.608 |
| 4.0 | 57.730 |

速度翻倍 → C·q̇ 严格 4 倍，验证 C·q̇ ∝ v² 关系。

### 实验 3: 反向-正向动力学一致性
随机生成 5 组 (q, q̇, q̈)，反向算 τ，再用 q̈ = M⁻¹(τ - bias) 还原。
最大误差 5.11e-15（机器精度）。

### 实验 4: 重力补偿曲线
joint2 从 -1.3 到 1.3 rad 扫描，最大重力力矩 32.5 N·m。

## 实现要点

### MuJoCo API 选择
- `mj_fullM`: 提取惯性矩阵 M
- `mj_rne(flg_acc=0)`: 提取 bias = C·q̇ + G

### 显式动力学公式
- 反向: τ = M @ q̈ + bias
- 正向: q̈ = M⁻¹ (τ - bias)
- 矩阵乘法在 numpy 显式完成
